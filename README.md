# KiraraBot
TAOのレア敵を通知するBOTです。

# メモ:
```
マイナンバー親の許可がもらえなかったのでこの部分だけ載せます
動作確認よろしく 不要な部分削っただけだから多分動く
```

# インストール:
```
パッケージをインストール
config.tomlにtokenとprefix書き込む
-python run.py
```
